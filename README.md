# 知乎日报-vue
开发框架：vue2.0(vue.js.vue-router) 构建工具: webpack

## 功能介绍
* 查看最新消息
* 查看主题列表
* 查看主题内容
* 查看文章详情
### tips 
部分内容受官方限制，无法查看。
## 使用
```
    node .\server\app.js
```
另启cmd或powershell，执行以下命令
```
  cd .\zhihu-daily-vue\
    yarn add|npm install|cnpm install
    yarn run dev | npm run dev |cnpm run dev  
```
